STAS 4.1 – Persistent Entry Page

- App öffnet bei jedem normalen Start auf der Einstiegsseite.
- Mann/Frau, Level und finales Ziel sind direkt auswählbar.
- Bestehende Nutzer können ihr Dashboard mit einem Button öffnen.
- Der Start-Button in der Navigation führt immer zur Einstiegsseite.
- Session-Ende führt ebenfalls zur Einstiegsseite zurück.
- Cache-Version: 4.1 / v410.
